# 🧁 Página de Receita - Cupcake de Café

![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=for-the-badge&logo=html5&logoColor=white)
![CSS3](https://img.shields.io/badge/CSS3-1572B6?style=for-the-badge&logo=css3&logoColor=white)

> **Projeto desenvolvido durante o curso Full Stack da Rocketseat**

Uma página web elegante e responsiva para apresentação de receitas culinárias, com design clean e moderno inspirado em cafeterias artesanais.

---

## 📋 Sobre o Projeto

Este projeto apresenta uma receita de **Cupcake de Café com Chantilly** em uma interface visualmente atraente, utilizando HTML semântico e CSS moderno. O layout foi cuidadosamente projetado para proporcionar uma experiência de leitura agradável e imersiva.

### ✨ Características

- **Design Responsivo**: Layout centralizado e adaptável
- **Tipografia Elegante**: Fonte Alice (Google Fonts) para melhor legibilidade
- **Paleta de Cores Harmoniosa**: Tons terrosos e aconchegantes (#573A37, #291B1A, #F0E8C2)
- **Imagens de Alta Qualidade**: Background e imagem principal otimizadas
- **HTML Semântico**: Estrutura bem organizada com tags apropriadas
- **CSS Moderno**: Uso de variáveis CSS e boas práticas de estilização

---

## 🚀 Tecnologias Utilizadas

- **HTML5**: Estrutura semântica e acessível
- **CSS3**: Estilização com Flexbox, Border-radius, Box-sizing
- **Google Fonts**: Fonte Alice para tipografia personalizada

## 🖥️ Como Visualizar

### Método 1: Abrindo diretamente no navegador

1. Clone ou baixe este repositório
2. Navegue até a pasta do projeto
3. Abra o arquivo `index.html` no seu navegador

### Método 2: Usando o Live Server (VS Code)

1. Instale a extensão **Live Server** no VS Code
2. Abra a pasta do projeto no VS Code
3. Clique com o botão direito no `index.html`
4. Selecione **"Open with Live Server"**
